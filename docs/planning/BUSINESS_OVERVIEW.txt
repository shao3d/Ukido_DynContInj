EXECUTIVE SUMMARY
-----------------
Интеллектуальный чат-бот, который автоматизирует первичные продажи и 
консультации для онлайн-школы soft skills Ukido. Система обрабатывает 95% 
входящих запросов без участия человека, увеличивая конверсию в пробные 
занятия на 40%.


БИЗНЕС-ПРОБЛЕМА
---------------
• 60% запросов приходят вне рабочих часов → теряются горячие лиды
• Операторы тратят 80% времени на одинаковые вопросы о ценах и расписании
• Тревожные родители требуют эмпатичного подхода, не терпят шаблонных ответов
• Стоимость лида $15-20, но конверсия в покупку только 8-12%


НАШЕ РЕШЕНИЕ
------------

Ключевые возможности:
• 24/7 консультирование с человеческой эмпатией
• Адаптивный тон под эмоциональное состояние родителя (4 психотипа)
• Умная воронка продаж с персонализированными CTA
• Сохранение контекста между сессиями (до 7 дней)
• Стриминг ответов — пользователь видит печать в реальном времени

Технологическая архитектура:
  Запрос → AI Router (Gemini) → AI Generator (Claude) → Ответ
             ↓                        ↓
        Классификация           База знаний (13 документов)
        намерения               + Адаптация тона

• Двухступенчатая AI архитектура экономит 40% на API
• Локальная обработка социальных запросов (приветствия, благодарности)
• Персистентность диалогов через JSON с автоочисткой
